Use an editorial minimalist visual style for the app.

The palette should be warm monochrome: cream backgrounds, soft black text, graphite gray metadata, very light warm-gray dividers, and a muted stone-beige accent. Avoid saturated colors. The app should feel like a premium writing tool, not a colorful productivity app.

Use a high-contrast serif font for large editorial moments such as the app concept title, greeting text, and empty states. Use a clean neutral sans-serif for all functional UI: navigation, note lists, metadata, buttons, settings, labels, and status text. For the Markdown editor, use a readable monospace or system text font depending on mode.

Suggested font pairing:

* Display serif: Canela, Editorial New, Playfair Display, or Cormorant Garamond.
* UI sans-serif: SF Pro Text, Inter, Geist, or Satoshi.
* Markdown/code: SF Mono, IBM Plex Mono, or JetBrains Mono.

Suggested colors:

* Background: #FAF8F3
* Surface: #FFFDF8
* Primary text: #111111
* Secondary text: #4A4A46
* Muted text: #8A8780
* Divider: #DDD8CE
* Dark overlay: #111211
* Muted accent: #C9C2B8

Keep typography restrained. Status bar text and icons should be small and lightly spaced. Phone UI text should feel realistic, not oversized. Use generous whitespace, thin separators, minimal icons, and subtle hierarchy.
